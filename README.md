# WiFiSwapper 🛰️

**WiFiSwapper** is a specialized Windows utility designed to give you total control over wireless network access using a unique **"Dual-Container"** system. The application allows for bulk activation and deactivation of Wi-Fi profiles, making it a powerful tool for access control and connectivity management.

---

## 🔄 How does it work?
The application operates using two management columns (containers):

* **Active List:** Network profiles currently registered in Windows, visible and ready to connect.
* **Inactive List:** "Frozen" profiles, stored within the app but removed from the Windows system connection stack.

With a single click of the **Swap** button, you can flip the contents of both lists. If you secure the process with a password, restoring access to "hidden" networks will require authorization.

---

## ✨ Key Features

* **Bulk Environment Management:** No more clicking through system settings – switch entire network environments at once.
* **Built-in Editor:** Full support for operations on both lists:
    * **Add:** Manually enter new SSIDs.
    * **Edit:** Modify parameters of existing profiles.
    * **Delete:** Clean up unused connections.
* **Password Lockdown:** Effective protection against unauthorized list swapping and network restoration.

## 🛠️ Installation & Requirements

1.  Download the latest version from the [Releases](#) section.
2.  **Run as Administrator:** Necessary for modifying Windows network settings.
3.  Set a password in the settings to activate Secure Mode.

## 📅 Roadmap
Planned features for upcoming releases:

* **Scheduling:**
    * Automated swaps at specific times (e.g., turn off home Wi-Fi at 10 PM and restore at 7 AM).
    * "Focus Mode" timer – temporary network cutoff (e.g., for 60 minutes).
* **UI Customization:**
    * Dark Mode support.
    * Custom naming for lists (e.g., "Work", "Relax", "Blocked").
* **Logs & Reports:**
    * History of swap attempts (including failed password entries).
    * System notifications on connection status changes.
* **Remote Trigger:** Remote swap triggering via a web panel or mobile app.
