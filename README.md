WiFiSwapper 🛰️
WiFiSwapper to narzędzie dla systemu Windows, które daje Ci pełną kontrolę nad dostępem do sieci bezprzewodowych poprzez system "dwóch kontenerów". Aplikacja pozwala na masowe aktywowanie i dezaktywowanie całych list sieci Wi-Fi, co czyni ją potężnym narzędziem do kontroli dostępu i zarządzania profilami łączności.

🔄 Jak działają listy?
Aplikacja opiera się na dwóch kolumnach zarządzania:

Lista Aktywna: Profile sieciowe, które są aktualnie zarejestrowane w systemie Windows i gotowe do połączenia.

Lista Nieaktywna: Profile "zamrożone", przechowywane w aplikacji, ale usunięte z systemowego stosu połączeń.

Za pomocą jednego przycisku Swap, możesz zamienić zawartość obu list miejscami. Jeśli proces zabezpieczysz hasłem, odblokowanie dostępu do "ukrytych" sieci będzie wymagało autoryzacji.

✨ Kluczowe funkcje
Zarządzanie Pełnymi Listami: Nie bawisz się w pojedyncze kliknięcia – przełączasz całe środowiska sieciowe naraz.

Wbudowany Edytor: Pełne wsparcie dla operacji na każdej z list:

Dodawanie: Ręczne wprowadzanie nowych sieci (SSID).

Edycja: Modyfikacja parametrów istniejących profili.

Usuwanie: Oczyszczanie list z nieużywanych połączeń.

Blokada Hasłem: Skuteczna ochrona przed nieautoryzowaną zamianą list i przywróceniem dostępu do sieci.

🛠️ Instalacja i wymagania
Pobierz najnowszą wersję z sekcji Releases.

Uruchom program jako Administrator (niezbędne do modyfikacji ustawień sieciowych Windows).

Zdefiniuj hasło w ustawieniach, aby aktywować tryb bezpieczny.

🗺️ Roadmap (Plany rozwoju)
Oto funkcje, które planujemy wprowadzić w nadchodzących wersjach:

📅 Harmonogramy (Scheduling)
Automatyczna zamiana list o określonych godzinach (np. automatyczne wyłączanie sieci domowej o 22:00 i przywracanie o 7:00).

Timer "Focus Mode" – czasowe odcięcie sieci na np. 60 minut.

🌓 Personalizacja Interfejsu
Wsparcie dla Dark Mode (Tryb Ciemny).

Możliwość nazywania list (np. "Praca", "Relaks", "Blokada").

📊 Logi i Raporty
Historia prób zamiany list (z informacją o błędnych hasłach).

Powiadomienia systemowe o zmianie statusu połączeń.

📱 Remote Trigger
Możliwość zdalnego wyzwalania zamiany list (np. przez prosty panel webowy lub aplikację mobilną).
